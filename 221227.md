#### 01\_ 협업과 도구

1. TDD : 테스트 코드 먼저 개발
2. Code Review
3. Version control
   - Git: 스냅샷(변화) 저장
   - Perforce: Large file 지원
   - SVN(Subversion): 클라이언트-서버 모델, 파일 저장
4. Performance Test
   - nGrinder : 성능테스트 미리미리 하기
5. 정적분석:
   - 코드 돌리기 전에 검사해주는 것
   - SonarQube(무료), Coverity(비쌈)
6. 로그 Tail
7. 유틸리티 만들어서 써보기
   Mac:NFD, Window:NFC 글자 하나씩 써져서 파일 보내지는 문제 변환해서 전송 가능

#### WebRTC

Web Real-Time Communication
