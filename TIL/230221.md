## 230221 FE ajax, div 내용 수정

1. ajax로 api 호출하기

```js
$.ajax({
  type: "POST",
  url: "",
  dataType: "json",
  data: { name: "홍길동" },
  success: function (results) {
    alert(results.msg);
  },
  error: function (data) {
    alert("err: " + data);
  },
});
```

2. div 값 변경하기

```js
function setInnerHTML() {
  const element = document.getElementById("my_div");
  element.innerHTML = '<div style="color:blue">InnerHTML<div>';
}
```
